GIT

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

  • sistema de controlo de versões
  • gratuito
  • open source
  • rápido
  • sistema distribuido, ou seja, cada utilizador pode ter uma cópia do repositório localmente
  • é o mais utilizado

comando para ver se temos instualado e qual a versão:git --version

para iniciar um repositório local git init

para consultar o status dos ficheiros: git status

para adicionar ficheiro ao staging area /index /área de preparação: git add intro_git.html

git add ficheiro 1 ficheiro 2

git add .

quando for para guardar uma versão do programa / fazer commit / efetuar snapshot:

git commit - m "pequena descrição"

para ver o historico de commits: git log

git log --oneline -graph

ramificação do código

listar os ramos/branch em qual estamos git branch

criar uma nova ramificação / branch git branch nomeramo

para trocar de ramo: git switch

para efetuar o merge git merge nomebranch
